# Color of Soul (COS) — 제품 요구사항 문서 (PRD)

**문서 버전:** 1.0  
**기준 저장소:** `cos` (Next.js App Router)  
**최종 갱신:** 2026-04-02  

---

## 1. 개요

### 1.1 제품 한 줄

**Color of Soul** 은 Studio COS 브랜드를 위한 **단일 페이지 랜딩 웹사이트**로, 명리·스펙트럼 기반 서비스 소개, 에너지 갤러리, 예약·연락 안내를 **글래스모피즘 카드**와 **Spline 3D 배경**으로 전달한다.

### 1.2 비즈니스 맥락

- **클라이언트:** Color of Soul (Studio COS)
- **기획·디자인·개발 협업:** [Pole Star Labs](https://www.polestar-labs.ai/)
- **저작권·라이선스:** 저장소 `LICENSE` 및 루트 `README.md` 의 Pole Star Labs 귀속 조항을 따른다.

### 1.3 제품 목표

| 목표 | 설명 |
|------|------|
| **브랜드 전달** | “빛의 언어”, 파장·에너지 등 브랜드 서사를 시각·카피로 일관되게 표현 |
| **정보 구조** | Main → About → Service → Gallery → Book → Contact 로 한 페이지 내 탐색 |
| **신뢰·협업 표기** | 푸터에 Color of Soul × Pole Star Labs 협업 및 링크 노출 |
| **배포 가능성** | 정적/Node 호스팅에 적합한 Next.js 빌드 산출물 |

## 2. 사용자·사용 시나리오

### 2.1 주요 사용자

- 브랜드와 서비스를 **처음 조사**하는 방문자
- **갤러리·예약·연락처** 정보가 필요한 잠재 고객
- 모바일·데스크톱 모두에서 **스크롤 탐색**하는 사용자

### 2.2 핵심 시나리오

1. **랜딩:** 첫 화면에서 인트로(“COLOR OF soul”) 후 메인 카피(영·한)가 순차 등장한다.
2. **섹션 이동:** 좌측 고정 내비(대형 화면) 또는 모바일 햄버거 메뉴로 앵커 이동한다.
3. **스크롤 탐색:** About, Service, Gallery, Book, Contact 를 순서대로 읽는다.
4. **외부 연동:** Pole Star Labs 링크, 예약·연락에 필요한 정보 확인(구체적 예약 시스템 연동은 제품 범위 외일 수 있음).

## 4. 비기능 요구사항

### 4.1 성능

- Spline iframe 은 백그라운드 탭에서 GPU 부하를 줄이도록 설계(가시성 API 등).
- `next/image` 로 갤러리 등 이미지 최적화; `sizes` 등 반응형 힌트 제공.
- 다중 `backdrop-blur` 는 저사양·모바일에서 비용이 클 수 있으므로 필요 시 브레이크포인트로 조절 가능하도록 문서화.

### 4.2 접근성·품질

- 의미 있는 `title` (iframe), `aria-label` / `aria-expanded` (모바일 메뉴), `role="contentinfo"` (푸터) 등 기본 ARIA 패턴.
- 키보드 포커스: `focus-visible` 아웃라인(링크·버튼).
- 메타데이터: `layout.tsx` 의 `title`, `description`, 파비콘.

### 4.3 국제화·콘텐츠

- 기본 언어: 한국어 중심 + 영문 보조 카피 병기(히어로·About 등).
- 법적·브랜드 문구는 클라이언트·협업사 합의에 따른다.

### 4.4 호환 환경

- Node.js 20+ 권장.
- 최신 Chromium·Safari·Firefox 및 모바일 브라우저에서 레이아웃·스크롤 동작 검증.
